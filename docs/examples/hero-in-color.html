Boot Colorful

Using the colors is easy. Add a color and a bg-color to the class attribute of a well or hero-unit, and watch it pop!

<div class="well bg-orange white">

Boot Colorfuller

If you want, you can even apply different color classes to most visible elements, but be careful you don't go overboard.

<h2 "class="blueDark">

You got this

If you can use Bootstrap, you can probably guess most of it.

This came from a really simple desire that went unfulfilled. Wow, it sounds overly-deep when I put it that way.

Basically, while I was working with Bootstrap on a web application, I wanted to change the color of something, and for whatever reason I assumed I could type the name of one of the colors which I had seen in the variables.less file (you may think that this was some unusual feat of memory, but seriously the colors are named pink and blue after all - not very hard to remember). Anyway, if you're familiar with bootstrap you probably know that I was off-base, and that you can't do that.

All you have to do is use the names of the Bootstrap primary color variables from the variables.less file in your tags - there's a handy list just below, but you probably won't need it after 5 minutes.
